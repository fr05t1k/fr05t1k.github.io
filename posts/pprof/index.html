PPROF. Profiling Goalng Applications.

2019-09-21
#golang  #doc 

Setup

Go Tools provide quite a lot of built-in instruments out of the box. Today we’re going to talk about one of them.

Import default pprof http endpoint:

import (
	_ "net/http/pprof"
)

If you don’t use http in your application you can just run it standalone:

package main

import (
	"net/http"
	_ "net/http/pprof"
)

func main () {
    http.ListenAndServe(":8081", nil)
}
